# Scardice-ui

> 余烬核心的前端实现

## 说明

本仓库为余烬核心的前端实现，使用 Vue3 + ElementPlus + tailwindcss 开发。

后端仓库：[Scardice-core](https://github.com/Scardice/Scardice-core)

此仓库的 Issue 已关闭，请在后端仓库中提交 Issue。包括与前端相关的 Issue。

## 部署

```bash
pnpm install
```

### 开发环境

```bash
pnpm run dev
```

### 生产环境

```bash
pnpm run build
```
